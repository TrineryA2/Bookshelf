# Bookshelf

**An offline Android app for managing your personal book collection with a 3D wooden shelf interface.**

Bookshelf is a lightweight Android application built with vanilla JavaScript and wrapped as an APK. Store your books, track reading status, and view them beautifully arranged on a wooden shelf. All data stays offline on your device.

---

## 📚 Features (v1.0 – Complete)

- ✅ **3D wooden shelf UI** – Your books are displayed on a realistic, interactive wooden shelf.
- ✅ **Add books easily** – Enter title, author, choose a cover color, or upload a custom cover image.
- ✅ **Track reading status** – Mark books as *Reading*, *Not Started*, or *Complete*.
- ✅ **Swipe through shelves** – Navigate your collection with a simple left/right swipe.
- ✅ **Searchable library** – Tap *My Books* to view every entry in a clean, searchable list.
- ✅ **Fully offline** – No internet required. All data is stored locally on your device.
- ✅ **Lightweight APK** – Small file size, optimized for Android 5.0+.

---

## 🚀 Upcoming Features (v2.0)

- **IndexedDB support** – Upgrade the storage engine to allow unlimited books and cover images.

- Web-to-App APK wrapper (Android)

## 📱 Platform

- **Android 5.0+** (ARM / ARM64)
- **File Size:** ~10.5 MB

---

## 📦 How to Install

1. Download the latest `.apk` file from the **Releases** page of this repository.
2. Transfer the file to your Android device.
3. Open the file and allow installation from unknown sources if prompted.
4. Launch Bookshelf and start adding your books!
